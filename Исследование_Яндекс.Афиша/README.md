# Исследование данных сервиса Яндекс Афиша

## Описание проекта

### Цель исследования
Провести исследовательский анализ данных сервиса Яндекс Афиша для выявления инсайтов об изменении пользовательских предпочтений и популярности событий осенью 2024 года, а также проверить гипотезы о разнице в поведении пользователей с мобильными и стационарными устройствами.

### Исследуемые данные
Для анализа были предоставлены три датасета:
* **final_tickets_orders_df.csv** — информация о заказах билетов
* **final_tickets_events_df** — данные о событиях
* **final_tickets_tenge_df.csv** — курс тенге к российскому рублю

### Основные этапы исследования
* Загрузка и предобработка данных
* Исследовательский анализ
* Статистический анализ
* Формулировка выводов и рекомендаций

### Использованные библиотеки
* **Pandas** — для работы с данными
* **NumPy** — для математических операций
* **Matplotlib** — для визуализации
* **Seaborn** — для построения графиков
* **SciPy** — для статистического анализа
* **Statsmodels** — для проверки гипотез

## Основные выводы

### Поведенческие паттерны пользователей
* **Мобильный сегмент** доминирует на рынке (70% заказов)
* Наблюдается устойчивый рост всех показателей
* Осенний период характеризуется увеличением активности пользователей

### Финансовые показатели
* **Средняя выручка** с заказа составляет 625 рублей
* **Медиана** — 355 рублей
* Основной диапазон покупок: 116-809 рублей

### Сегментация по устройствам
* **Мобильные пользователи**:
  * Средняя выручка: 610 рублей
  * Среднее количество билетов: 2.3

* **Десктопные пользователи**:
  * Средняя выручка: 650 рублей
  * Среднее количество билетов: 2.5

* Увеличить маркетинговые активности в категории театр и концерты

## Подтвержденные гипотезы
* Средняя выручка с заказа отличается для мобильных и десктопных устройств
* Среднее количество билетов в заказе различается в зависимости от типа устройства
* Наблюдается сезонность в поведении пользователей

Проект демонстрирует значительный потенциал для дальнейшего развития сервиса и оптимизации пользовательского опыта на основе полученных инсайтов.
